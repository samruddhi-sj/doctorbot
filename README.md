# MyDoctor Bot

This is my Graduation Project ;)  
My Doctor bot is a diseases diagnosis web application made with flask-python. (See the screenshots)

## Technologies used:

* HTML5
* CSS
* JQuery
* Bootstrap v3.3
* python
* flask (python framework)
* Machine Learning (Decision tree algorithm)

## Used Libraries

* font-awesome
* rwdImageMaps.js
* nicescroll.js
* Google Fonts
